# To-Do-List API

A robust and simple API for managing personal tasks. This project showcases essential API development practices, including user authentication and CRUD operations for tasks.

## Features

This API provides the following core functionalities:

*   **User Authentication**: Secure user registration and login utilizing [JSON Web Tokens (JWT)](https://jwt.io) for authorization.
*   **Task Management**: Comprehensive [CRUD (Create, Read, Update, Delete)](https://wikipedia.org) operations for managing tasks.
*   **Task Status Tracking**: Functionality to mark tasks as `complete`.
